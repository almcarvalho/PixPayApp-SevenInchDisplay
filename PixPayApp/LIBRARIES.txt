// Required Libraries for PixPayApp
// Install these through Arduino IDE -> Sketch -> Include Library -> Manage Libraries

1. WiFiManager by tzapu
   - Version: 0.16.0 or later
   - Purpose: WiFi configuration with captive portal

2. GFX Library for Arduino by moononournation  
   - Version: 1.3.7 or later
   - Purpose: High-performance graphics for ESP32 RGB displays

3. lvgl by kisvegabor
   - Version: 8.3.x (NOT 9.x)
   - Purpose: GUI framework for embedded systems

4. QRCode by Richard Moore
   - Version: 0.0.1 or later  
   - Purpose: QR code generation for PIX payments

5. ESP32 Board Package
   - Install via Boards Manager
   - Add URL: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   - Search: ESP32 by Espressif Systems
